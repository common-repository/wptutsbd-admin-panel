== Description ==
WpTutsBD Admin Panel plugin is for Styling Your Own WordPress Dashboard Panel and its absolutely free .It allows users to easily customize the look of the administration interface for WordPress . Now with this plugin you can re-brand your Websites back-end too.

Key Features: (Light version)

* Nice n Clean design 
* 5 color schemes: Blue (default), Green, Red, Yellow, Purple 
* Custom login Page (Pro Version)
* Custom logo or Text 
* Choose your Menu icons (Pro Version)
* Easy installation 
* Multilanguage support  ( Pro version )

Features Pro version:

* Custom Colors
* Custom Css panel
* Admin logo
* Custom login Page
* Option hover to show submenu
* Custom dashboard widget
* Close menu when open another
* Enable HTML in footer text
* Choose your Menu icons (Pro Version)
* Footer rebranding
* Multilanguage support  ( .PO .MO included )

== Installation ==

1. Upload the `/Wptutsbd_admin` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Navigate to `Dashboard->WPtutsBD Admin` and Start Customizing.
